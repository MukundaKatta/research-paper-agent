# research-paper-agent

Stub Python project. Described as an AI research paper agent, but none of that functionality exists.

## What's actually here

A single class in `src/core.py` (ResearchPaperAgent) with placeholder methods that each return a fixed dictionary without doing any real work. There is no paper search, no citation analysis, no literature review automation - just stubs.

The project uses only Python standard library imports (time, logging, json, typing). No academic or NLP libraries are present.
